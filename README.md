# Ellis Collective House

A Next.js + TypeScript starter for the Ellis Collective House website. The project is preconfigured with Tailwind CSS, Shadcn UI utilities, and Framer Motion for immersive interactions.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI primitives
- Framer Motion
